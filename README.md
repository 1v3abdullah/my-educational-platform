# My Educational Platform

A dynamic, secure, and modern full-stack educational platform built with **Python & Flask**. This web application is tailored to provide students with a seamless, responsive, and data-driven learning experience.

---

## Features Implemented

### 1.  Authentication & Session Management
- Secure user login and session handling via Flask sessions.
- Dynamic routing ensuring authenticated users only access the internal portal.

### 2.  Comprehensive Student Profile (`/profile`)
- **Profile & Account:** Responsive forms to update username, contact details, and password.
- **Learning Activity:** Interactive dashboard showcasing total study hours and badges earned.
- **Finance & Subscription Management:** Premium tier indicators, saved payment methods, and an interactive invoice tracking system.

---

##  Tech Stack

- **Backend:** Python, Flask, Gunicorn
- **Frontend:** HTML5, CSS3 (Custom responsive layouts, CSS variables, grid system), FontAwesome Icons

##  How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python backend_login.py
   ```
